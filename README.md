## Technology Stack

- **Framework**: Next.js 16 with React 19
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **UI Components**: shadcn/ui
- **Fonts**: Playfair Display (headings), Inter (body)
- **Analytics**: Vercel Analytics
- **Internationalization**: Custom i18n implementation (EN/VI)
- **Icons**: Lucide React
- **Theming**: Light/Dark mode with system preference detection

## Getting Started

### Installation

1. Clone the repository
\`\`\`bash
git clone https://github.com/thinkingschool/thinkingschool-canada.git
cd thinkingschool-canada
\`\`\`

2. Install dependencies
\`\`\`bash
npm install
\`\`\`

3. Set up environment variables
\`\`\`bash
cp .env.example .env.local
\`\`\`

4. Run development server
\`\`\`bash
npm run dev
\`\`\`

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Deployment

Deploy to Vercel with one click:

\`\`\`bash
npm run build
npm start
\`\`\`
